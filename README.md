# react-native-rich-chat-input

A high-performance React Native input component that supports native rich content (GIFs, stickers, and images) via Android's OnReceiveContentListener and iOS's Pasteboard API.

## Installation


```sh
npm install react-native-rich-chat-input
```


## Usage


```js
import { RichChatInputView } from "react-native-rich-chat-input";

// ...

<RichChatInputView color="tomato" />
```
